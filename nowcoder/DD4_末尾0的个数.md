# myCode
- 传统迭代：46ms  4556KB
```python
import sys

line = sys.stdin.readline().strip()
n = int(line)
m = 1
for i in range(n):
    x = i + 1
    m *= x
s = str(m)
s=s[::-1] # 倒序
num = 0
for i in s:
    if i == '0':
        num += 1
    else:
        break
print(num)
```
- %10：45ms	4652KB
```python
import sys

line = sys.stdin.readline().strip()
n = int(line)
m = 1
for i in range(n):
    x = i + 1
    m *= x
num = 0

while m % 10 == 0:
    num += 1
    m = m // 10
print(num)
```
- 阶乘函数：43ms	4600KB
```python
import sys
import math

line = sys.stdin.readline().strip()
n = int(line)
m = math.factorial(n)
num = 0

while m % 10 == 0:
    num += 1
    m = m // 10
print(num)
```
