# 题干
- 描述  
小东和三个朋友一起在楼上抛小球，他们站在楼房的不同层，  
假设小东站的楼层距离地面N米，球从他手里自由落下，  
每次落地后反跳回上次下落高度的一半，并以此类推直到全部落到地面不跳，  
求4个小球一共经过了多少米？(数字都为整数)  
给定四个整数A,B,C,D，请返回所求结果。

# 示例
- 测试样例：  
100,90,80,70
- 返回：  
1020

# 核心代码模式
- 输出格式
```python
return xxx
```

等比数列(1/2+1/4+...+1/n)求和为1
